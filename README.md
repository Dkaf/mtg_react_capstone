# Magic the Gathering Deck Builder App

A simple deck builder for the Magic the Gathering card game utilizing mtg API

## Installing
```
> git clone https://github.com/Dkaf/mtg_react_capstone.git
> npm install
```

## Launching
Dev Mode
```
> npm start
```
Production
```
> npm build
```
